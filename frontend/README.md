# Pothole Detection & Reporting System

A smart web application that helps citizens identify and report potholes.  
The system allows users to view pothole locations, upload images, and contribute information that can help authorities maintain roads efficiently.

---

## 🚀 Features

- View reported potholes
- Image-based contributions
- Track newly added reports
- Responsive UI
- Fast development build with Vite

---

## 🛠 Tech Stack

- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Vite
